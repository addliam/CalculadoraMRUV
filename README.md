# CalculadoraMRUV
Debes tener en cuenta la Distancia (d), Tiempo (t), Velocidad inicial (Vo), Velocidad final (Vf) y Aceleracion (a).
Completa los valores que tengas, deja en blanco las variables que seran calculadas.
